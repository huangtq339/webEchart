# webEchart
echart is used to visualize the data and display them through line charts, bar charts, pie charts, etc

The following knowledge is required to complete the project:

 ① div + css layout
 ② flex layout
 ③ Less
 ④ Native js + jquery use
 ⑤ rem adaptation
 ⑥ echarts Foundation
